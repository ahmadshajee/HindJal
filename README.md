# Hind Jal

Premium Next.js website for the Hind Jal water business.

## What is included

- Glass UI landing page with founder-led storytelling
- Product catalog with premium retail cards
- Secure checkout and contact flow backed by MongoDB
- Mobile-first navigation and layout

## Run locally

```bash
npm install
npm run dev
```

## Environment

Set `MONGODB_URI` and `MONGODB_DB` in `.env.local`.
